## Usage/Access Information

-   **Description:** a program that should be able to handle the spell check of words that are contained i nthe dictionary.txt.
        using hashing, and several imports, it should be able to handle
	cases of mispelling, close enough words are offered a certain
	of suggested words, if the suggested word is not right should 
	return false. it should be true if yes is replied at any point.
        
-   **Prerequisites:** cpp complier, cpp extension
-   **Installation/Setup:** dependent on your system, see screen recording for execution expected on a windows machine
-   **Instructions for Use:** 
            debug and run hashingDict. collect input through user input
	and display true or false and alternative options if a certain 
	case is met
-   **Known Issues/Limitations:** does not handle really really short words
